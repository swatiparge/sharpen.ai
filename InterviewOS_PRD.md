# InterviewOS PRD

**Tagline:** Grammarly for Interviews\
**Positioning:** The Career Performance Analytics Platform for
Candidates. Track. Improve. Measure.

------------------------------------------------------------------------

## 1. Executive Summary

InterviewOS is an AI-powered Career Performance Analytics Platform
designed for job candidates.

It records and analyzes real interview conversations (audio/screen),
extracts structured signals, evaluates performance quality independent
of hiring politics, detects long-term patterns, and builds a
personalized improvement roadmap.

The system functions as an **Interview Performance Operating System
(OS)** --- enabling continuous performance tracking and measurable
growth.

------------------------------------------------------------------------

## 2. Problem Statement

Candidates face repeated rejections without meaningful feedback.

### Key Problems:

-   No structured feedback after interviews
-   Cannot remember all follow-up questions
-   "I did well" vs actual performance gap
-   No longitudinal performance tracking
-   No clarity on skill gaps blocking next-level roles
-   Random, unstructured improvement attempts
-   Confidence drops due to ambiguity

The market is employer-centric. No platform deeply analyzes
candidate-side performance over time.

## 7. Core Features

### 7.1 Interview Recording Engine

-   Audio recording
-   Screen recording (optional)
-   Auto transcription
-   Speaker segmentation
-   Question/answer segmentation

### 7.2 Interview Memory Reconstruction AI

-   Structured post-interview prompts
-   Guided recall workflow
-   Voice-based reconstruction
-   AI-assisted question inference

### 7.3 Signal Extraction Engine

Extract structured data: - Question type - Follow-up depth - Rebuttal
handling - Tradeoff discussion presence - Quantification usage - Clarity
of explanation - Confidence markers - Seniority-level signals -
Technical correctness probability - Creative solution reasoning

System must: - Detect incorrect understanding - Avoid hallucinations -
Not penalize creative solutions - Understand level expectations

### 7.4 Role-Specific Intelligence

Evaluation calibrated by role: - Frontend - Backend - Product Manager -
UX Designer - DevOps

Each role has its own evaluation rubric.

### 7.5 Performance Metrics Framework

Scoring Dimensions: 1. Communication Clarity 2. Structural Thinking 3.
Technical Depth 4. Tradeoff Awareness 5. Quantification & Impact 6.
Follow-up Handling 7. Seniority Alignment 8. Confidence Signal Strength

All scores must be explainable.

### 7.6 Interview Skill Graph

Dashboard showing: - Skill dimension scores - Trend over time -
Improvement velocity - Weak vs strong categories

### 7.7 Interview Gap Analyzer

Detect recurring weaknesses across interviews and provide: - Pattern
summary - Severity score - Impact analysis

### 7.8 Company-Specific Mode

Calibrated evaluation styles: - FAANG - Startup - Consulting -
Leadership-heavy interviews

### 7.9 Performance Benchmarking

Compare user against: - Role-level expectations - Experience band
expectations - Top-performer baselines

### 7.10 Weakness-Driven Adaptive Simulation

Generate targeted mock interviews based on detected weaknesses and
increase difficulty progressively.

### 7.11 Career Level Calibration

Provide Level Gap Report: - Current performance level - Target level
comparison - Missing signals - Required improvements
